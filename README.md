# Emergency Communications Analytics

## Overview

This project demonstrates an end-to-end business intelligence solution built using Snowflake and Power BI. It includes data loading, SQL transformations, dimensional modeling, reporting views, and an interactive dashboard for monitoring emergency communications performance.

## Tools

- Snowflake
- SQL
- Power BI
- DAX
