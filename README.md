# Emergency Communications Analytics

## Overview

Emergency Communications Analytics is an end-to-end business intelligence project designed to simulate reporting for an emergency communications center.

The project follows a typical BI workflow by loading operational data into Snowflake, transforming it with SQL, building a dimensional model, and developing an interactive Power BI dashboard to monitor call volume, service performance, and operational KPIs.

---

## Business Scenario

Emergency communication centres rely on timely and accurate reporting to monitor call demand, response performance, and service levels.

This project demonstrates how operational call data can be transformed into meaningful dashboards that support performance monitoring and day-to-day decision making.

## Dashboard Metrics

The dashboard includes reporting for:

- Total Calls
- Service Level
- Average Answer Time
- Average Handle Time
- Abandonment Rate
- Transfer Rate
- Call Volume Trends
- Calls by Region
- Calls by Priority
- Calls by Type
- Channel Distribution
